# 靜態資產說明

部署前請將實際資產放置於對應目錄：

| 目錄 | 用途 | 格式/規格 |
|------|------|----------|
| `hero/` | 主視覺圖片 | `hero.jpg`（WebP 建議，≥1920px 寬） |
| `story/` | 愛情故事圖片 | `2019.jpg`、`2021.jpg`、`2024.jpg`、`2026.jpg` |
| `gallery/` | 相片牆 | `photo-1.jpg` … `photo-6.jpg`（可自行調整數量） |
| `music/` | 背景音樂 | `bg-music.mp3`（建議 < 5MB） |
| `og/` | Open Graph 分享圖 | `og-cover.jpg`（1200×630px） |
| `line/` | LINE QR Code | `qr-code.png`（建議 512×512px） |
| `ui/` | 介面通用圖示 | 視設計需求 |
